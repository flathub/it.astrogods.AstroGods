Flatpak manifest for [AstroGods](https://astrogods.it), a modern Astrology app focused on self-knowledge. It provides in-depth birth chart interpretations using Egyptian astrology (Equal house system with Vehlow domification), synastry analysis for relationships, and daily and monthly transit readings.

Source code: [arcangelo7/astrogods-app](https://github.com/arcangelo7/astrogods-app)
